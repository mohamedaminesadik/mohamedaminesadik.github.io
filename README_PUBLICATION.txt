PORTFOLIO - MOHAMED AMINE SADIK

Contenu du dossier :
- index.html : page principale du portfolio
- assets/ : photo, QR LinkedIn et CV PDF
- robots.txt : autorise l'indexation Google
- sitemap.xml : fichier à adapter avec votre URL finale

AVANT PUBLICATION :
1. Remplacer dans index.html : https://votre-domaine.com/ par l'URL finale du site.
2. Remplacer dans robots.txt et sitemap.xml : https://votre-domaine.com/ par l'URL finale du site.
3. Vérifier si vous voulez vraiment rendre publics votre téléphone et votre e-mail.

OPTION RECOMMANDÉE POUR UN SITE HTML : GITHUB PAGES
1. Créer un compte GitHub.
2. Créer un repository nommé par exemple : mohamed-amine-sadik-portfolio.
3. Uploader tout le contenu de ce dossier, surtout index.html à la racine.
4. Aller dans Settings > Pages.
5. Choisir la branche main et le dossier /root.
6. Attendre la génération du lien public.

POUR APPARAÎTRE SUR GOOGLE :
1. Ouvrir Google Search Console.
2. Ajouter votre URL comme propriété.
3. Utiliser l'outil URL Inspection.
4. Demander l'indexation de la page.
5. Attendre le crawl de Google. L'apparition dans Google n'est pas immédiate ni garantie le même jour.

REMARQUE :
Google Sites permet de créer et publier un site via l'éditeur Google, mais il n'est pas idéal pour héberger directement ce fichier HTML personnalisé. Pour ce portfolio, GitHub Pages ou Netlify est plus adapté.
